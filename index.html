Living off the Land (LotL) em TI: Uma Ameaça Discreta e Eficiente

O que é Living off the Land (LotL)?

"Living off the Land" (LotL) é uma técnica usada por cibercriminosos e agentes de ataques avançados para se infiltrar em sistemas sem depender de malware tradicional. Em vez disso, os invasores utilizam ferramentas e processos legítimos já presentes no sistema operacional ou em softwares confiáveis para realizar atividades maliciosas. Essa abordagem dificulta a detecção, pois as ferramentas usadas são consideradas normais pelo sistema de segurança, permitindo que os atacantes operem de forma discreta e persistente.

Como Funciona o Ataque LotL?

Os invasores exploram binários e scripts nativos do sistema, como:

Windows: PowerShell, WMI (Windows Management Instrumentation), PsExec, certutil.exe, BITSAdmin.

1. Linux: Bash, Python, Cron jobs, SSH.

2. Ferramentas de administração: Sysinternals, RDP (Remote Desktop Protocol), ferramentas de rede como Netcat.

3. Esses recursos, originalmente destinados a tarefas administrativas, são manipulados para:

4. Coleta de dados (exfiltrar informações sensíveis).

5. Movimentação lateral (avançar para outras máquinas na rede).

6. Execução remota de comandos.

7.Persistência (manter acesso prolongado ao sistema).

Por que LotL é Perigoso?

1. Dificuldade de Detecção: Como as ferramentas são legítimas, soluções de antivírus e EDR (Endpoint Detection and Response) podem não identificar o comportamento como malicioso.

2. Minimização de Artefatos: Não há necessidade de baixar arquivos suspeitos, reduzindo alertas de segurança.

3. Ataques Persistentes: Técnicas LotL são comuns em ataques de APTs (Advanced Persistent Threats) e ransomware.

como se proteger contra Living off the land

Exemplos de Técnicas LotL

1. PowerShell Malicioso: Uso de scripts PowerShell para baixar e executar payloads.

2. Abuso do WMI: Agendamento de tarefas maliciosas via WMI para persistência.

3. Certutil.exe para Download: Uso do utilitário de certificados do Windows para baixar malware.

4. Living off Trusted Sites: Hospedagem de scripts maliciosos em serviços legítimos (GitHub, Dropbox).

Conclusão

1. Living off the Land é uma ameaça sofisticada que explora a confiança em ferramentas do sistema, tornando a detecção desafiadora. Organizações devem adotar estratégias proativas, como monitoramento avançado e políticas de segurança rigorosas, para mitigar riscos. Ataques LotL mostram que, em segurança cibernética, até o que é legítimo pode ser usado contra você.
